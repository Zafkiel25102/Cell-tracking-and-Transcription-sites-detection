
# ***Cell tracking and Transcription sites detection***

<br><br>

## Set up conda virtual environment
<details>
<summary><b>Installing environment dependencies on Linux :</b></summary>

```yaml
# Enter to the main folder
cd cell_track

# create tracking conda environment python=3.8 pytorch==1.8.0 torchvision==0.9.0 cudatoolkit=11.1 faiss-gpu pytorch-lightning==1.4.9
conda create --name cell_track --file condarequirements_gnn.txt
conda activate cell_track

# install other tracking requirements
pip install -r piprequirements_gnn.txt
```
</details>

## Structure
<details>
<summary><b>The directory structure :</b></summary>

```
├── cell-tracker-gnn-main      
│   ├── configs           <- Hydra configuration files
│   ├── data              <- Tracking Project data
│   ├── logs              <- Logs generated by Hydra and PyTorch Lightning loggers
│   ├── notebooks         
│   ├── outputs           <- Outputs generated by deep metric learning model
│   ├── src               <- GNN Structure
│   ├── src_metric_learning     <- Metric_Learning Structure            
│   
├── segment-anything      
│   ├── assets            <- Example Imgs
│   ├── demo              <- Segment Anything Simple Web demo
│   ├── notebooks         
│   ├── scripts              <- ONNX Export
│   ├── segment_anything     <- SAM Structure     
│
├── README.md               <- All information
├── cell_track 
│   ├── condarequirements_gnn.txt      <- File for installing python dependencies
│   ├── piprequirements_gnn.txt        <- File for conda environment creation
│   ├── condarequirements_sam.txt      <- File for installing python dependencies
│   ├── piprequirements_sam.txt        <- File for conda environment creation
│   ├── bash        <- main folder
│   │    ├── cell-tracker-gnn-main      
│   │    │  ├── configs           <- Hydra configuration files
│   │    │  ├── data              <- Tracking Project data
│   │    │  ├── logs              <- Logs generated by Hydra and PyTorch Lightning loggers
│   │    │  ├── notebooks         
│   │    │  ├── outputs           <- Outputs generated by deep metric learning model
│   │    │  ├── src               <- GNN Structure
│   │    │  ├── src_metric_learning     <- Metric_Learning Structure            
│   │    ├── segment-anything      
│   │    │  ├── assets            <- Example Imgs
│   │    │  ├── demo              <- Segment Anything Simple Web demo
│   │    │  ├── notebooks         
│   │    │  ├── scripts              <- ONNX Export
│   │    │  ├── segment_anything     <- SAM Structure   
│   │    ├── cell_track_yolo.sh      <- Main Bash Script   
  
```
</details>
